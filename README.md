## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
    
   git clone https://github.com/ganeshnitalikar/Expense-Tracker.git
     
2. Navigate to the project directory:
    
   cd expense_tracker
     
3. Install dependencies:
    
   flutter pub get
     
4. Run the app:
    
   flutter run
